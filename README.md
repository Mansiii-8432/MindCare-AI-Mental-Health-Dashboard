## üîß Customization

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
  --primary-color: #A78BFA;
  --secondary-color: #EC4899;
  --accent-color: #60A5FA;
}
```

### Modify AI Responses
Edit response patterns in `script.js`:
```javascript
// In Chatbot.generateResponse() function
if (message.includes('your-keyword')) {
  return "Your custom response";
}
```
